# attendance_app

Features :

1. Role Based Authentication
2. QR code Attendance
3. Firebase Cloud Messaging
4. Firebase Cloud Storage
5. Firebase Firestore
6. Permissions
7. Read And Write to the Device Storage
8. Notifications
9. Live Location Tracking
10. Google Map SDK
